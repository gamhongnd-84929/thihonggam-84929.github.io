Skype Pros And Cons

  • Pros

    1. Screen sharing option available through Skype
    2. Time setting 24/7
    3. Easy to install
    4. Group call without the need for group features
    5. Subscription fee is cheap
    6. Safe calling device
    7. Upgrade frequently

  • Cons

    1. No access to emergency service
    2. No real face-to-face interaction
    3. No language translation service
    4. Sound qualities are based on bandwidth
    5. Information could be intercepted
